# Personal AI Tracker

An AI-powered personal tracking and self-improvement system that analyzes daily behavior, generates personalized insights, and provides automated daily motivation and monthly improvement reviews using GenAI and long-term memory.

## Features
- Daily personal data tracking
- Automated analytics
- Daily AI-generated motivational guidance
- Monthly AI-driven self-improvement reports
- Retrieval-Augmented Generation (RAG) with vector memory
- Secure JWT authentication

## Tech Stack
- FastAPI
- PostgreSQL + pgvector
- Celery + Redis
- GenAI (LLM APIs)
